为什么换到 Viel Frame

类型: yapping | CREATED 2026-06-07

这个站点没有使用任何博客工具或站点生成器。每一篇文章都是完全纯手写的 HTML 文件。

这种方式一开始很简单，可是，谁都能看出来，维护成本很高。

但对于我来说，这些博客工具什么的完全不值得，也许以后会去学吧。

现在我做出了 Viel Frame。它是一个 CSS 框架，不需要写任何类页面就能自动获得样式。

这就意味着我可以继续写纯静态 HTML。新增文章只需要复制一个极简模板，把内容填进去就行。而且CDN链接我用的是最新npm版本（通常不建议用于生产环境，但我就要用）。我也不用更新这个链接了。
